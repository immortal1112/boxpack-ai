# BoxPack AI · 智能装箱系统

## 快速开始
1. 把整个 `boxpack` 文件夹复制到任意电脑
2. 用 Chrome 打开 `index.html`
3. 点击地址栏右侧「安装」按钮，添加到桌面
4. 之后直接点桌面图标使用，离线可用

## 文件说明（维护必看）

| 文件 | 用途 | 谁来改 |
|------|------|--------|
| `skudb.js` | SKU尺寸数据库 | 仓库主管，记事本直接编辑 |
| `boxdb.js` | 箱型数据库 | 仓库主管，记事本直接编辑 |
| `packing.js` | 装箱算法核心 | 开发者 |
| `history.js` | 历史记录模块 | 开发者 |
| `print.js` | 打印箱唛模块 | 开发者 |
| `sw.js` | 离线缓存控制 | 更新版本时改版本号 |
| `index.html` | 主界面 | 开发者 |

## 新增SKU（记事本操作）
打开 `skudb.js`，在最后一个 SKU 后面加一行：
```
'新SKU编号': { name:'品名', l:长, w:宽, h:高, wt:重量 },
```
保存后刷新页面即可。

## 新增箱型
打开 `boxdb.js`，复制现有一行修改：
```
{ id:'STD-02', name:'小箱', l:40, w:30, h:25, maxWt:15 },
```

## 更新发布（局域网用户）
1. 修改对应文件
2. 打开 `sw.js`，把 `boxpack-v1` 改成 `boxpack-v2`
3. 其他电脑刷新页面自动同步
